The project can be divided into several steps:

Step 1: Camera Calibration

Camera calibration is the process of determining the intrinsic and extrinsic parameters of a camera. Intrinsic parameters include the focal length, principal point, and distortion coefficients. Extrinsic parameters include the position and orientation of the camera relative to the world coordinates. In this step, we use a chessboard pattern to calibrate the camera.

Read More

Step 2: Marker Detection

In this step, we detect the Aruco Marker in the image using OpenCV. We first detect the corners of the marker and then use the unique code embedded in the marker to identify it. We then use the camera matrix and distortion coefficients obtained from the camera calibration to estimate the pose (position and orientation) of the marker relative to the camera.

Read More

Step 3: Object Detection

In this step, we detect the object whose size we want to measure. We first capture an image without the object and then capture an image with the object. We then subtract the two images to obtain the image of the object alone. We then use OpenCV to detect the contours of the object.


Read More

Step 4: Size Measurement

In this step, we measure the size of the object using the pose of the marker obtained in step 2 and the contours of the object obtained in step 3. We first project the 3D coordinates of the corners of the marker onto the image plane using the pose. We then use the distance between the corners of the marker in the image and the known physical size of the marker to estimate the pixel-to-meter conversion factor. We then use this conversion factor to convert the size of the object in pixels to the size in meters.
