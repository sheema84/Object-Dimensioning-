About Us

Object size measurement is a common task in many fields, such as manufacturing, robotics, and computer vision. Accurately measuring the size of an object is important for quality control, automation, and analysis. Traditional methods of object size measurement involve the use of rulers or calipers, which can be time-consuming and prone to human error. In recent years, computer vision has emerged as a powerful tool for object size measurement.

OpenCV is a popular open-source computer vision library that provides various functions and tools for image processing, object detection, and tracking. Aruco Marker is a kind of marker that is widely used in computer vision and augmented reality. It contains a unique code that can be easily detected and recognized by the computer vision algorithm. In this project, we will use OpenCV and Aruco Marker to measure the size of an object accurately and efficiently.
